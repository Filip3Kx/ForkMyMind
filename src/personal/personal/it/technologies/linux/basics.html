Archiwowanie

Example (Archiwowanie logów jenkinsa):

Tar

tar -czvf nazwa.tar.gz /var/log/jenkins

tar -xzvf nazwa.tar.gz -C ścieżka

Zip

zip -r nazwa.zip /var/log/jenkins

unzip -x nazwa.zip

Filtry

GREP:

  • Ignorowanie case'a: -i
  • Rekursywne szukanie: -R
  • Negacja: -v

Doł pliku z odświeżaniem: tail -f

CUT:

  • Separator: -d
  • Pole: -f 
cat /etc/passwd | cut -d: -f1

AWK <---koks:
awk -F':' '{print $1}'
  • Filtrowanie samych błędów: 2>>
  • Filtrowanie samego outputu: 1>>
  • Filtrowanie wszystkiego: &>>

Procesy

  • top
  • ps

Stany:

  • running
  • sleeping
  • stopped
  • zombie

Proces Zombie - Proces który nie został zakończony mimo że wykonał wszystkie zadania

Zabicie wszystkiego z tekstem httpd

ps -ef | grep httpd | grep -v 'grep' | awk '{print $2}' | xargs kill -9

Vim

  • :se nu - wyświetlnie numerów lini
  • : - pójście do jakiejś lini
  • gg - początek pliku
  • G - koniec pliku
  • yy - kopia lini
  • 4yy - kopia 4 lini pod kursorem
  • pp -wklejka lini
  • dd - usuwanie i kopia do schowka
  • u - undo
  • / - search
  • ^n - nastepne wystąpienie
